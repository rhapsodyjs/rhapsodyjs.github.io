Introduction

Hello there! So you're giving RhapsodyJS a try, you won't be disappointed, I guarantee! =)

As your first step I would suggest you to read how Controllers work in RhapsodyJS, you'll see that they are self-routed, so you won't have to create a lot of route files, just the file with the actions and the info about the controller and its subcontrollers and views folder, in a organized and hierarchical way.

You're not understanding something, you can just ask on RhapsodyJS repository on Github.

Hey, and if you want to contribute with the project, feel free to create a pull request. =)
